# Emoji Test Suite 👍 😄 
Assessing Emoji Use in Modern Text Processing Tools

### Results available for the following tasks:
- Tokenization
- Part of speech tagging
- Dependency Parsing
- Sentiment analysis

### List of tools used in the experiments:
- Gensim
- NLTK
- NLTK Tweet Tokenizer
- PyNLPl
- SpaCy
- SpaCyMoji
- Stanford CoreNLP
- Stanza
- Textblob
